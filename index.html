פונקציות מחרוזות

סיכום פונקציות על מחרוזות

len()

מחזירה את אורך המחרוזת

len("hello") → 5

upper() / lower()

המרה לאותיות גדולות/קטנות

"Hello".upper() → "HELLO"

strip()

הסרת רווחים מהתחלה והסוף

" hello ".strip() → "hello"

split()

פיצול המחרוזת לרשימה

"a,b,c".split(",") → ["a","b","c"]

join()

חיבור רשימה למחרוזת

",".join(["a","b"]) → "a,b"

replace()

החלפת טקסט במחרוזת

"hello".replace("l","x") → "hexxo"

find() / index()

מציאת מיקום תו או מחרוזת

"hello".find("e") → 1

startswith() / endswith()

בדיקה אם מתחיל/נגמר במחרוזת

"hello".startswith("he") → True
