יסודות התכנות

📝 הכרת שפת פייתון

  • מה זה פייתון ולמה משתמשים בו?
  • כתיבת התוכנית הראשונה
  • הפעלת קוד פייתון
  • תכנות אינטראקטיבי
print("שלום עולם!")
# זוהי ההדפסה הראשונה שלנו

🔤 משתנים

  • מה זה משתנה?
  • כללי מתן שמות למשתנים
  • השמת ערכים
  • שינוי ערכי משתנים
name = "יוסי"
age = 25
height = 1.75

📥 קלט ופלט

  • פונקציית print() - הדפסה למסך
  • פונקציית input() - קבלת קלט מהמשתמש
  • עיצוב הפלט
  • המרת טיפוסים בקלט
name = input("מה השם שלך? ")
print("שלום", name)

טיפוסי נתונים

🔢 מספרים

  • מספרים שלמים (int)
  • מספרים עשרוניים (float)
  • פעולות חשבון בסיסיות
  • המרה בין טיפוסי מספרים
x = 10          # int
y = 3.14        # float
result = x + y  # 13.14

📝 מחרוזות (str)

  • יצירת מחרוזות
  • הדבקה והשרשור
  • אורך מחרוזת
  • גישה לתווים
text = "פייתון"
print(len(text))  # 5
print(text[0])    # פ

✅ בוליאניים (bool)

  • ערכים: True ו-False
  • השוואות והשוואות לוגיות
  • המרה לבוליאני
  • שימוש בתנאים
is_student = True
age = 18
can_vote = age >= 18  # True

📋 רשימות (list)

  • יצירת רשימות
  • הוספה והסרה של איברים
  • גישה לאיברים
  • מתודות רשימה שימושיות
fruits = ["תפוח", "בננה", "תפוז"]
fruits.append("ענבים")
print(fruits[0])  # תפוח

אופרטורים

➕ אופרטורים חשבוניים

  • חיבור (+), חיסור (-)
  • כפל (*), חילוק (/)
  • חילוק שלם (//), שארית (%)
  • חזקה (**)
a = 10
b = 3
print(a + b)   # 13
print(a // b)  # 3
print(a % b)   # 1

🔍 אופרטורי השוואה

  • שווה (==), לא שווה (!=)
  • גדול מ (>), קטן מ (<)
  • גדול או שווה (>=)
  • קטן או שווה (<=)
x = 5
y = 10
print(x == y)  # False
print(x < y)   # True

🔗 אופרטורים לוגיים

  • and - "וגם"
  • or - "או"
  • not - "לא"
  • עדיפות אופרטורים
age = 20
is_student = True
discount = age < 25 and is_student

📝 אופרטורי השמה

  • השמה רגילה (=)
  • השמה עם חשבון (+=, -=)
  • השמה עם כפל (*=)
  • השמה עם חילוק (/=)
x = 10
x += 5    # x = x + 5
x *= 2    # x = x * 2
print(x)  # 30

הוראות ובקרה

🔀 תנאים (if)

  • הוראת if בסיסית
  • else - במקרה הפוך
  • elif - תנאים מרובים
  • תנאים מקוננים
grade = 85
if grade >= 90:
    print("מעולה")
elif grade >= 80:
    print("טוב מאוד")
else:
    print("צריך להשתפר")

🔁 לולאת for

  • מעבר על רשימות
  • שימוש ב-range()
  • לולאות מקוננות
  • break ו-continue
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num * 2)

for i in range(3):
    print(f"ספירה: {i}")

🔄 לולאת while

  • לולאה עם תנאי
  • מונה ועדכון תנאי
  • הימנעות מלולאה אינסופית
  • שימוש ב-break
count = 0
while count < 5:
    print(f"ספירה: {count}")
    count += 1

🔧 פונקציות

  • הגדרת פונקציה עם def
  • פרמטרים וארגומנטים
  • החזרת ערכים עם return
  • פונקציות עם ערכי ברירת מחדל
def greet(name, greeting="שלום"):
    return f"{greeting}, {name}!"

message = greet("יוסי")
print(message)  # שלום, יוסי!

מחרוזות בפייתון

📖 מה זו מחרוזת?

מחרוזת היא רצף של תווים (אותיות, מספרים, סימנים). במחרוזת אנחנו יכולים לשמור טקסט, שמות, הודעות ועוד.

דוגמאות בסיסיות:

name = "יוסי"           # עם גרשיים כפולים
city = 'תל אביב'         # עם גרש בודד
message = "שלום עולם!"   # מחרוזת עם רווחים
empty = ""              # מחרוזת ריקה

🔧 פונקציות מחרוזות עיקריות

len() - אורך המחרוזת

מחזירה את מספר התווים במחרוזת

len("שלום") → 4
len("Hello World") → 11

upper() - אותיות גדולות

הופך את כל האותיות לגדולות

"hello".upper() → "HELLO"
"Python".upper() → "PYTHON"

lower() - אותיות קטנות

הופך את כל האותיות לקטנות

"HELLO".lower() → "hello"
"Python".lower() → "python"

count() - ספירת תווים

סופר כמה פעמים תו מופיע במחרוזת

"hello".count("l") → 2
"python".count("n") → 1

📋 אינדקסים ובדיקות

כל תו במחרוזת יש לו מקום (אינדקס) שמתחיל מ-0:

text = "Python"
print(text[0])      # P (תו ראשון)
print(text[1])      # y (תו שני)
print(text[-1])     # n (תו אחרון)

# בדיקה אם טקסט נמצא במחרוזת
if "th" in text:
    print("נמצא!")   # ידפיס כי "th" נמצא ב-"Python"

💡 דוגמאות שימושיות

🔤 השוואת מחרוזות:

name1 = "יוסי"
name2 = "יוסי"
if name1.lower() == name2.lower():
print("זהה!")

📊 סטטיסטיקה על מחרוזת:

text = "Hello Python"
print(f"אורך: {len(text)}")
print(f"כמות o: {text.count('o')}")

� בדיקת תוכן:

email = "user@gmail.com"
if "@" in email:
print("זה נראה כמו אימייל")
